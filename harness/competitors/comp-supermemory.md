## Competitor
MCP memory server + plugins for Claude Code / OpenCode, ~$2.6M seed (Google/Cloudflare angels),
~21.7K GitHub stars. Strong LongMemEval scores (85.2%).

## Positioning
Personal knowledge-management vault; deep Claude Desktop integration; "best for solo creators and
knowledge workers." Free with $5/mo usage, then pay-as-you-go. This is the **closest competitor to
Ruksak's solo-first user**.

## Gap Ruksak can exploit
PKM/recall framing centred on Claude Desktop, not cross-platform product-context for builders
shipping software. Again memory-not-decisions, no governed write path, no harness/strategy layer,
weaker multi-agent (Codex/Cursor/Windsurf) parity than its Claude focus suggests.
