## Competitor
Best-funded memory layer (~$24M). OpenMemory is its shared MCP memory server for Cursor, VS Code,
Claude, and any MCP tool. Dual vector + graph mode, universal SDK.

## Positioning
"Universal memory layer for AI agents & apps." Free tier (10k memories), Pro $19–249/mo, Enterprise.
Optimised for semantic recall and accuracy benchmarks.

## Gap Ruksak can exploit
Stores **memories** (facts, embeddings) for recall, not **decisions/reasoning with governance and
status**. Developer-infra framing (SDK-first), less a curated product-context surface for a solo
builder. No structured strategy/harness layer, no proposal-backed review of writes.
